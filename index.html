Hungry Chameleon

An interactive game where user’s control the orientation of a chameleon trying to catch all the moving flies. The point is for the user to control the state of the chameleon with arrow keys for rotation and the space bar for catching the flies (chameleon with tongue out). If the chameleon is touched by a fly without its tongue being out then the game is over.

How to download


Clone the project repository to your local machine.
git clone https://github.com/olincollege/softdes-hungry-chameleon-.git

Navigate to the project directory:
cd hungry-chameleon

Install the required packages and libraries by running the following command:
pip install -r requirements.txt

Run the game by executing main.py:
python main.py

OR

Run the game by executing python3 in the terminal of the project directory:
python3 hungry_chameleon/
